About the System

AI-based CV analysis platform that leverages machine learning to assist recruiters with data-driven decisions.

Intelligent Talent Analytics

Our system extracts key information from resumes, generates technical questions from GitHub profiles, predicts attrition risk using historical patterns, and provides market value benchmarking for candidates.

  • GitHub-based question generation for technical screening
  • Early attrition risk prediction using ML models
  • Real-time market value assessment based on skills & experience

Project Domain

Literature Survey

Recent studies highlight the use of NLP and transformer models for resume parsing. GitHub-based question generation has emerged as a novel approach for technical screening. Attrition prediction using XGBoost and random forests shows promising accuracy. Market value assessment remains fragmented across salary databases. This research integrates all three components into a unified decision-support system.

Research Problem

Traditional recruitment processes suffer from manual CV screening bias, lack of technical validation, inability to predict candidate attrition, and inconsistent compensation benchmarking. This leads to suboptimal hiring decisions and increased turnover costs.

Research Gap

Existing HR tech solutions address either CV parsing OR skill assessment, but none combine GitHub-based dynamic question generation, early attrition risk prediction, and real-time market valuation in a single platform.

Research Objectives

  • Develop a GitHub-based question generation module to assess technical skills.
  • Build an early attrition risk prediction model using candidate profile data.
  • Create a market value assessment engine using live salary benchmarks.
  • Integrate all modules into a web-based dashboard for recruiters.

Methodology

Agile methodology combined with CRISP-DM for data science pipeline. Data collection from resumes (PDF/DOCX), GitHub API, and salary databases. Feature extraction using NLP embeddings. Models: Random Forest for attrition, regression for market value, and rule-based + LLM for question generation. Deployment via Flask API and Bootstrap frontend.

Technologies Used

  • Python (Pandas, Scikit-learn, PyTorch)
  • Flask / FastAPI (Backend)
  • GitHub REST API
  • React / Bootstrap (Frontend)
  • AWS / Google Cloud (Deployment)
  • PostgreSQL / MongoDB
