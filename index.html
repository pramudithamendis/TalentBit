About the system

Intelligent Talent Analytics

Our platform is an AI-driven recruitment solution that extracts key information from resumes, generates technical screening questions from GitHub profiles, predicts attrition risk using historical patterns, evaluates candidates by integrating CV, LinkedIn, and GitHub data, identifies the most suitable job positions based on job requirements, and provides real-time market value benchmarking for candidates.

  • Multi-Source Candidate Evaluation based on the CV,GitHub and LinkedIn profiles.
  • Real-time market value assessment based on skills and experience
  • Early attrition risk prediction using ensemble ML models
  • GitHub-based dynamic question generation for technical screening
Research

Project Domain

Literature Survey

Recent studies highlight the use of NLP and transformer models for resume parsing. GitHub-based question generation has emerged as a novel approach for technical screening. Attrition prediction using XGBoost and random forests shows promising accuracy. Market value assessment remains fragmented across salary databases. This research integrates all three components into a unified decision-support system.

Research Gap

Existing HR tech solutions address either CV parsing or skill assessment, but none combine GitHub-based dynamic question generation, early attrition risk prediction, and real-time market valuation in a single integrated platform.

Research Problem

Traditional recruitment processes suffer from manual CV screening bias, lack of technical validation, inability to predict candidate attrition, and inconsistent compensation benchmarking — leading to suboptimal hiring decisions and increased turnover costs.

Research Objectives

  • Multi-Source Candidate Evaluation based on the CV,GitHub and LinkedIn profiles.
  • Create a market value assessment engine using live salary benchmarks.
  • Build an early attrition risk prediction model using candidate profile data.
  • Develop a GitHub-based question generation module to assess technical skills.
  • Integrate all modules into a web-based dashboard for recruiters.

Methodology

Agile methodology combined with CRISP-DM for the data science pipeline. Data collection from resumes (PDF/DOCX), GitHub API, and salary databases. Feature extraction using NLP embeddings. Models include Random Forest for attrition, regression for market value, and rule-based + LLM for question generation. Deployment via Flask API and Bootstrap frontend.

Technologies Used

  • Python — Pandas, Scikit-learn, PyTorch
  • Flask / FastAPI for backend services
  • GitHub REST API for profile analysis
  • React / Bootstrap frontend
  • AWS / Google Cloud deployment
  • PostgreSQL / MongoDB databases
